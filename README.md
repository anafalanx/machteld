# machteld

*Working name — a Windows 11 machine-control language. You write Tcl; power is added in C. Design stage; no code yet.*

A single signed Windows executable (Tcl/Tk 9 + C, one file) that exposes kernel-grade process control and Windows machine access as a small, self-describing command palette — built to be as legible to agents as to humans.
